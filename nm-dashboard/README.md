## 🚀 Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Lucide Icons, Socket.io-client
- **Backend**: Node.js (Express), Puppeteer, WhatsApp-web.js, Socket.io
- **Database**: Supabase (PostgreSQL + Auth)

## 🛠️ Getting Started

1. **Backend**: 
   ```bash
   cd backend
   npm start
   ```

2. **Frontend**:
   ```bash
   cd nm-dashboard
   npm run dev
   ```
